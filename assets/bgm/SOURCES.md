# BGMの出典

| ファイル | 出典 | ライセンス |
|---|---|---|
| ambient / mystery / warm / beat | 自作シンセ（`ytf assets --init` で再生成） | — |
| curious.mp3 ほか生成曲 | MusicGen（`ytf bgm` でローカル生成） | 生成物は自由利用 |
| amacha_*.mp3 | 甘茶の音楽工房 https://amachamusic.chagasi.com/ | フリー・商用可・クレジット任意 |
| bgmer_*.mp3 | BGMer https://bgmer.net/ | フリー・商用可・クレジット不要 |

他サイトの曲を使う場合もこの表に追記すること（DOVA等は曲ごとに規約確認）。
